# Conversor de Unidades en C

Este proyecto implementa un **conversor de unidades** desarrollado en lenguaje **C**, que permite realizar conversiones de **temperatura, longitud y peso** desde la línea de comandos.  

---

## Características principales
- Conversión de **temperatura** entre °C, °F y K.  
- Conversión de **longitud** entre metros, pies, centímetros y pulgadas.  
- Conversión de **peso** entre kilogramos, libras, gramos y onzas.  
- Interfaz de **menú interactivo** en consola.  
- Validación de entradas numéricas.  
- Código estructurado y comentado.

## Requisitos

- Compilador C (por ejemplo, GCC o Clang).
- Sistema operativo compatible con consola (Windows, Linux o macOS).

---

## Compilación

### En Linux o macOS:

1. Abre una terminal en la carpeta donde se encuentra el archivo.
2. Compila el programa con:

   ```bash
   gcc -o conversor_unidades conversor_unidades.c
   ```

### En Windows (usando MinGW o similar):

1. Abre CMD o PowerShell en el directorio del archivo.
2. Compila con:

   ```bash
   gcc -o conversor_unidades.exe conversor_unidades.c
   ```

---

## Ejecución

**Linux / macOS:**

   ```bash
   ./conversor_unidades
   ```

**Windows:**

   ```bash
   .\conversor_unidades.exe
   ```
